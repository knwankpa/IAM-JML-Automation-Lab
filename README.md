# IAM-JML-Automation-Lab

This repo contains an Identity & Access Management (IAM) Joiner/Mover/Leaver (JML) automation lab.

## What it includes
- **PowerShell** JML workflow simulation
- Mock HR data in `data/mock_hr_users.csv`
- Automation script in `scripts/jml_automation.ps1`
- `logs/` placeholder for audit logs

## Future enhancements
- Microsoft Graph integration
- Entra ID user provisioning and access management
- Governance/reporting dashboard
